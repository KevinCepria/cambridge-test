## Libraries Used

- Node v20.16
- Next 14.2.15
- React 18
- Tailwin CSS
- Typescript
- Headless UI
- Axios
- Reacht Hook Forms

## Existing Routes

- / - Login
- /articles - Articles owned by current logged in user
- /search - All articles

## To install (development)
1. cd to directory
2. Run `npm i`
3. Run `npm run dev`
4. Go to localhost:3000

## To install (production)
1. cd to directory
2. Run `npm i`
3. Run `npm run build`
4. Run `npm run start`
5. Go to localhost:3000
